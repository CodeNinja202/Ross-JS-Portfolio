Projects

Strangers Things

A digital implementation of Craigslist light. Logged in users are able to search items other users post, send the user a message, post their own items to sell, edit and delete their posts using programming concepts like components, API calls, and async functions.

Fitness Tracker

A React application designed to attract potential users in their quest to get healthy. Users are able to create a workout routine, add an activity to each routine and publicly post their routine if they desire. If a user is logged in, they are shown a tab to display their own personal routines.

Guessing Game

A fun and interactive guessing game created with HTML, Javascript, and CSS that allows the user to guess a number between 1-100. The game limits the user to 5 guesses and provides feedback after each guess until the correct number is guessed.

Arcade '86

Arcade 86' is a retro-style arcade game created using HTML, CSS, and Javascript. It features classic games such as Snake and Wack-a-mole.

Quote Generator

The quote generator built using React, Material UI, and fetch API's allows users to generate random quotes and share them on Twitter.

451 Books

Currently re-deploying the backend. An online bookstore shopping experience that allows a customer/visitor to search items currently in stock, add the item to a cart, edit/delete the cart and checkout. A logged in customer cart stays persistent to allow the customer to pick up where they left on multiple devices.
